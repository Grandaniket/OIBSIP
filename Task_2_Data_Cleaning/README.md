# Data Cleaning Project

## Overview
This project was completed as part of the Oasis Infobyte Data Analytics Internship (OIBSIP). The objective was to clean and preprocess a raw dataset by handling missing values, duplicates, inconsistent data and formatting issues to prepare it for further analysis.

## Problem Statement
Real world datasets often contain missing values, duplicate records and inconsistencies that can affect the accuracy of analysis. The goal of this project was to clean the dataset and improve its quality for reliable data analysis.

## Tools Used
1. Python
2. Numpy
3. Pandas
4. Matplotlib
5. Jupyter Notebook

## Project Workflow
1. Loaded the dataset
2. Standardized data formats
3. Identified missing values
4. Handled duplicate records
5. Corrected data inconsistencies
6. Saved the cleaned dataset
