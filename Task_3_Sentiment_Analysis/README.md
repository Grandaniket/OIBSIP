# Twitter Sentiment Analysis on Twitter Data

## Overview - 
The project was completed as part of the Oasis Infobyte Data Analytics Internship(OIBSIP). The objective was to implement Twitter Sentiment Analysis using Natural Language Processing (NLP) and Machine Learning. The project also includes data visualization, model evaluation and sentiment prediction on text.

## Problem Statement -
The project focuses on building a machine learning model that can automatically classify online tweets as Positive, Neutral or Negative, helping organizations analyze customer opinions and social media trends more efficiently.

## Tools Used -
1. Python
2. Pandas
3.  Matplotlib 
4.  Seaborn
5.  Scikit-learn

## Project Workflow -
1. Data loading
2. Data Cleaning
3. Splitting the data into training and testing sets.
4. Converting text into numerical feature using TF IDF.
5. Trained a Logistic Regression model.
6. Model evaluated using Accuracy Score, Classification Report and Confusion Matrix.

## Conclusion
The model successfully classified tweets into positive, neutral and negative sentiments with high accuracy of 91%.
