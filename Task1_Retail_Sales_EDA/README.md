# Exploratory Data Analysis (EDA) on Retail Sales Data 

## Overview :
This project was completed as part of the Oasis Infobyte Data Analytics Internship (OIBSIP). The objective was to perform Exploratory Data Analysis (EDA) on a retail sales dataset to identify patterns, trends and insights that support better business decisions.

## Problem Statement :
Retail businesses generate large volumes of sales data every day. This project focuses on cleaning, exploring and analyzing the dataset to uncover meaningful patterns, trends and insights.

## Tools Used :
1. Python
2. Pandas
3. Numpy
4. Matplotlib
5. Seaborn
6. Jupyter Notebook

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Descriptive Statistics
5. Data Visualization
6. Insights and Conclusions
