# Wine Quality Prediction using Machine Learning

## Overview -
This project focuses on predicting the quality of red wine using Machine Learning techniques. The dataset was preprocessed, explored through visualizations and used to train multiple classification models. The performance of each model was evaluated and compared to identify the best performing algorithm.

## Problem Statement -
Wine quality is influenced by several chemical properties, making manual quality assessment difficult and time consuming. The objective of this project is to build a Machine Learning model that predicts wine quality accurately.

## Tools Used -
1. Python
2. Pandas
3. NumPy
4. Matplotlib
5. Seaborn
6. Scikit-learn

## Project Workflow -
1. Loaded and explored the wine quality dataset.
2. Performed data preprocessing and feature selection.
3. Visualized the data.
4. Split the dataset into training and testing sets.
5. Standardized the feature values using StandardScaler.
6. Trained three Machine Learning models, SGD Classifier, SVC and Random Forest Classifier
7. Evaluated each model using Accuracy Score, Classification Report and Confusion Matrix.
8. Compared the performance of all three models and choose the most accurate one.

## Conclusion -
Random Forest Classifier model achieved the highest accuracy of approx. 69%, outperforming the SGD and SVC models.
